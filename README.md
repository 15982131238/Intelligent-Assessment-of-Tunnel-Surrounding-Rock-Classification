# 隧道围岩等级智能评估系统

针对隧道掌子面图像质量退化与特征表达单一性问题，提出一种融合多尺度渐进式增强与深度语义建模的围岩智能评估方法。

## 主要功能

- **多尺度图像增强**：通过空域滤波与变换域去噪，信噪比提升12.7 dB，显著增强裂隙、节理等关键地质结构的可见性
- **四维综合评价指标体系**：基于灰度共生矩阵、改进型局部二值模式和RGB颜色矩，实现岩体状态量化表征
- **改进ResNet架构**：集成多尺度特征提取和双重注意力机制（通道注意力+空间注意力）
- **加权交叉熵-标签平滑复合损失函数**：有效解决类别不平衡问题

## 项目结构

```
├── annotations/         # 标注数据
├── tunnel_face_images/  # 隧道掌子面原始图像
├── processed_tunnel_face_photos/  # 处理后的图像
├── 智能预测模型（围岩等级）/  # 模型文件
├── 节理裂隙/            # 节理裂隙相关图像
└── 湿润/                # 湿度相关图像
```

## 技术栈

- Python 3.8+
- PyTorch/TensorFlow
- OpenCV
- NumPy
- SciPy
- Scikit-learn

## 使用方法

1. 准备隧道掌子面图像数据
2. 运行图像预处理脚本
3. 加载预训练模型
4. 进行围岩等级预测

## 注意事项

- 本项目使用的是公开数据集进行模型训练
- 实际应用中需要根据具体工程条件进行参数调整
- 模型仅作为辅助评估工具，最终围岩等级确定需结合工程经验
